# Sales Data Analysis Project
  This project analysis sales data to identify trends, top-performing  products, and regional performance.
## Tools used
  - Python
  - Pandas
  - Matplotlib
## Key Insights
  - Sales  are highest in month  of December
  - Technology category performs best
  - West region leads in sales
## Files Included
  - sales_analysis.py
  - SuperStoreOrders.xlxs
  - cleaned_sales_data.xlsx
